## 🚀 Gitee Pages 部署步骤

### 第1步:注册Gitee
1. 访问 https://gitee.com
2. 注册账号(可用微信登录)

### 第2步:创建仓库
1. 登录后,点击右上角 "+" → "新建仓库"
2. 填写信息:
   - 仓库名称:随便起,比如 `my-portfolio`
   - 勾选"公开"
   - 不需要勾选其他选项
3. 点击"创建"

### 第3步:上传文件
1. 进入刚创建的仓库
2. 点击"上传文件"
3. **重要:**
   - 先上传 `index.html` 文件
   - 再点击"新建文件夹",输入 `images` 并确认
   - 进入 `images` 文件夹,上传所有图片

### 第4步:开启Gitee Pages
1. 在仓库页面,点击"服务" → "Gitee Pages"
2. 点击"启动"按钮
3. 等待几秒,会生成访问链接
4. 链接格式: `https://你的用户名.gitee.io/仓库名`

### 第5步:测试
1. 复制生成的链接
2. 在浏览器打开测试
3. 用微信扫码测试能否正常打开

## ⚠️ 注意事项

1. **文件名必须是 `index.html`** - 这样访问时不需要加文件名
2. **images文件夹和HTML在同一目录** - 保持相对路径正确
3. **每次修改后** - 需要在Gitee Pages页面点击"更新"按钮
4. **首次访问可能较慢** - 等待30秒-1分钟

## 🔧 常见问题

**Q: 图片显示不出来?**
A: 检查images文件夹是否和index.html在同一级目录

**Q: 修改后没效果?**
A: 在Gitee Pages页面点击"更新"按钮,并清除浏览器缓存

**Q: 微信打不开链接?**
A: Gitee Pages在微信中完全支持,如果打不开可能是网络问题
